# KeywordTracker - Free SEO Analytics App

A Progressive Web App (PWA) that tracks top search keywords, analyzes traffic patterns, and provides SEO insights.

## Features
- Real-time keyword tracking
- Traffic analytics charts
- Regional traffic data
- Works offline
- Installable on any device
- 100% free, no registration

## Deployment Options

### Option 1: GitHub Pages (Recommended)
1. Create a GitHub account
2. Create a new repository named `keyword-tracker`
3. Upload all files to the repository
4. Go to Settings > Pages
5. Select "main" branch as source
6. Save - Your app will be available at `https://yourusername.github.io/keyword-tracker`

### Option 2: Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your project folder
3. Get a free domain (e.g., yourname-keywordtracker.netlify.app)
4. Automatic HTTPS included

### Option 3: Google Drive
1. Save the HTML file
2. Upload to Google Drive
3. Share with "Anyone with link"
4. Users can open directly in browser

## Usage
- Open the app in any browser
- Click "Install App" to add to home screen
- Works on mobile, tablet, and desktop
- No internet required after first load

## Privacy
- All data is simulated
- No data is sent to any server
- Everything runs locally in browser
- No tracking or analytics
