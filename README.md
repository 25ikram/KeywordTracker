# KeywordTracker - Free SEO Analytics App

A Progressive Web App (PWA) that tracks top search keywords, analyzes traffic patterns, and provides SEO insights.

## Features
- Real-time keyword tracking
- Traffic analytics charts
- Regional traffic data
- Works offline
- Installable on any device
- 100% free, no registration
